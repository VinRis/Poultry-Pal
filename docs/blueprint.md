# **App Name**: Poultry Pal

## Core Features:

- Learning Modules: Provide comprehensive learning modules on layers, broilers, different breeds, and poultry farming techniques.
- Disease Diagnosis: Allow users to upload images of sick chickens or droppings. The AI tool will analyze the images to suggest possible diseases and treatments. Note: This is not medical advice.
- Disease Library: A disease library containing symptoms and cures.
- Breed Information: Provide details on local breeds such as Kienyeji, Kuroiler, and Kenbro.
- Layers Guide: Guide users on eggs production and housing.
- Broilers Guide: Guide users on meat production tips and housing.

## Style Guidelines:

- Primary color: Vibrant lime green (#32CD32), conveying freshness, growth, and health related to farming.
- Background color: Light, desaturated green (#F0FFF0), providing a clean and calming backdrop.
- Accent color: A bright yellow (#FFD700), used for interactive elements to guide user action, to indicate the vibrancy and warmth.
- Body and headline font: 'PT Sans' (sans-serif) for clarity and readability.
- Use clear, intuitive icons representing different aspects of poultry farming and disease diagnosis.
- Maintain a clean and organized layout with a focus on easy navigation for both beginner and experienced farmers.
- Employ subtle animations for feedback, such as loading states or successful diagnosis, to improve user experience.
